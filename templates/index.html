О нашем проекте

Целью нашего проекта является помощь пользователям в изучении английского языка.

Наш сайт написан на языке Python с использованием фреймворка Django.

Он является улучшенной версией нашего прошлогоднего проекта, который был desktop приложением.

Мы перенесли наш проект в WEB формат для удобного использования.
